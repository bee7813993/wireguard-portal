# 利用ガイド — WireGuard Portal

自宅・社内の PC で動いている Web サーバーを、インターネットに公開するための手順です。

---

## 必要なもの

| 項目 | 内容 |
|------|------|
| PC の OS | Windows 10 / 11 |
| ソフトウェア | [WireGuard for Windows](https://www.wireguard.com/install/)（無料） |
| Web サーバー | XAMPP など、ポート **80** で動作するもの |
| ポータル URL | 管理者から共有された URL |

---

## STEP 1 — ポータルで設定ファイルを作成する

1. 管理者から共有されたポータルの URL をブラウザで開きます。
2. 「**公開ポート番号**」に、使用するポート番号（例: `11090`）を入力します。  
   ※ ポート番号は管理者に確認してください。他の利用者と重複できません。
3. 「**設定を生成する**」をクリックします。
4. 「**↓ wg-client_11090.conf をダウンロード**」をクリックしてファイルを保存します。

> 同じポート番号で再生成すると以前の設定は無効になります。再インポートが必要です。

---

## STEP 2 — WireGuard をインストールする

まだインストールしていない場合のみ。

1. https://www.wireguard.com/install/ を開きます。
2. 「Windows Installer」をダウンロードして実行します。
3. 「次へ」を押し続けてインストール完了。

---

## STEP 3 — 設定ファイルをインポートする

WireGuard を起動すると最初は空の画面が表示されます。

![WireGuard 起動直後](images/step3-empty.png)

1. 画面中央の「**ファイルからトンネルをインポート**」をクリックします。  
   ※ または左下「**トンネルの追加 ▼**」→「**ファイルからインポート**」
2. ダウンロードした `wg-client_11090.conf` を選択して「**開く**」をクリックします。
3. 左側のリストに `wg-client_11090` が追加されれば成功です。

![インポート完了後](images/step3-imported.png)

右側に以下の情報が表示されます。

| 項目 | 表示内容 |
|------|---------|
| 状態 | 無効 |
| アドレス | `10.100.10.x/24` |
| DNS サーバー | `1.1.1.1` |
| ピア / エンドポイント | `[サーバーアドレス]:51821` |

---

## STEP 4 — トンネルを有効化する

1. 左のリストで `wg-client_11090` を選択します。
2. 「**有効化(A)**」ボタンをクリックします。

![トンネル有効化後](images/step4-connected.png)

接続が成功すると：

- 左側のアイコンが **緑色のシールド** に変わります。
- 状態が「**有効**」になります。
- 数秒後に「**直近のハンドシェイク: ○秒前**」が表示されます。

> ハンドシェイクが表示されない場合は[トラブルシューティング](#トラブルシューティング)を参照してください。

---

## STEP 5 — Web サーバーを起動する

PC のポート **80** で Web サーバーを起動します。

**XAMPP の場合:**
1. XAMPP Control Panel を開く
2. 「Apache」の「**Start**」をクリック
3. ステータスが緑になれば OK

**XAMPP 以外の場合:** 使用している Web サーバーをポート 80 で起動してください。

---

## STEP 6 — 外部からアクセスして確認する

ブラウザで以下の URL にアクセスします：

```
http://[管理者から教えてもらったサーバーアドレス]:[ポート番号]
```

例: `http://example.com:11090`

スマートフォンや別の端末からアクセスして、自分の PC の Web サーバーが表示されれば完成です。

---

## PC を再起動したときは

トンネルは PC 再起動で切断されます。再び公開するには：

1. WireGuard を起動する
2. `wg-client_11090` を選択して「**有効化(A)**」をクリック
3. Web サーバーが起動していることを確認する

**自動で再接続したい場合：**  
WireGuard で `wg-client_11090` を右クリック →「**プロパティ**」→「コンピューターの起動時にトンネルを有効化」にチェック

---

## トラブルシューティング

### ハンドシェイクが表示されない

| 確認項目 | 対処 |
|---------|------|
| トンネルが「有効」になっているか | 「有効化(A)」を押す |
| Windows ファイアウォールがブロックしていないか | WireGuard をファイアウォールで許可する |
| 設定ファイルが古くなっていないか | ポータルで再生成してインポートし直す |

### 外部からアクセスしても自分のサイトが表示されない

1. WireGuard のトンネルが「**有効**」になっているか確認
2. Web サーバー（Apache 等）が **起動中** か確認
3. ポータルで **同じポート番号** を再度入力して設定を再生成
4. 新しい `.conf` ファイルを WireGuard に **上書きインポート**（古いトンネルを削除してから）

### 設定を上書きインポートする手順

1. WireGuard で古いトンネルを選択 → 左下の「**×**」で削除
2. ポータルで設定を再生成してダウンロード
3. 新しい `.conf` ファイルをインポートして有効化

## よくある質問

**Q. トンネルを有効にすると、普段のインターネットに影響しますか？**  
A. 影響しません。VPN サーバーへの通信だけがトンネルを通り、通常のブラウジングなどはそのまま使えます。

**Q. ポート番号を変えたい**  
A. 新しいポート番号でポータルから設定を再生成して、WireGuard にインポートし直してください。古いポートの設定削除は管理者に依頼してください。

**Q. ポート番号を忘れた**  
A. `.conf` ファイルのファイル名（例: `wg-client_11090.conf`）でポート番号を確認できます。または管理者にお問い合わせください。

**Q. 「ハンドシェイク」って何ですか？**  
A. PC と VPN サーバーが正常に通信できているかを示す指標です。「直近のハンドシェイク: ○秒前」と表示されていれば正常に接続されています。
